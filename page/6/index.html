Test cases

DP.

  1. Base case: dp[0] = 0
  2. Induction rule: dp[i] = min(dp[j] + 1) where j < i;

Time: O(n^3)
Space: O(n)

Dp.
Instead of check every value in (1, i), we only check the possible coin value, which is a much smaller range.
