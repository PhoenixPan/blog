279.Perfect Squares

Solution 1: accepted 48ms

  1. Base case: dp[0] = 0;
  2. Induction rule:
    dp[i] = min(dp[i - j j] + 1) where j j <= i
    dp[1] = dp[0] + 1 = 1;
    dp[2] = dp[1] + 1 = 2;
    dp[3] = dp[3] + 1 = 3;
    dp[4] = min(dp[4-11=3] + 1, dp[4-22=0] + 1) = 1;
